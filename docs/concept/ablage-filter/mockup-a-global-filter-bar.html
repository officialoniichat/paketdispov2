Wie das Filtern funktioniert: Quick-Chips und „Weitere Filter" wirken additiv (UND-Verknüpfung) auf alle Lanes gleichzeitig. Karten, die nicht (mehr) passen, verschwinden aus der Lane — die Lane selbst bleibt bestehen, auch wenn 0 Treffer übrig sind, damit die Fachlogik-Struktur (Status-Workflow) jederzeit erkennbar bleibt. Der Lane-Zähler zeigt „gefiltert/gesamt", sobald ein Filter aktiv ist. Das „Gruppieren nach"-Dropdown fügt optionale Sub-Header innerhalb jeder Lane ein (hier: nach Bereich) — das ist eine Verallgemeinerung der bestehenden Weiterleitungs-Empfänger-Gruppierung, keine neue Lane-Achse.
