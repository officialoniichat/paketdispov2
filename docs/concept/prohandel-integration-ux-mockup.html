Was der Pull füllt

Jedes Feld kommt strukturiert aus ProHandel — kein OCR, keine Schätzung.

Kopf

weBelegNo · Lieferant · Filiale · Buchungsdatum · Abschnitt · Catman · Verladetag

Lagerplatz

Lagerplatz-Code + Klasse → storageLocationId. Macht die Einlagerungs-Station überflüssig.

Position
